# iOS Mobile App

## Overview
Native iOS application for ProRAW/DNG image capture with real-time color card detection.

## Technology Stack
- Swift 5.9+, AVFoundation ProRAW, Core Image + OpenCV iOS
- SwiftUI + MVVM Architecture

## Features
- ProRAW/DNG capture with manual controls
- ArUco marker detection with Vision framework
- Background upload with URLSession
- Device capability detection

## Requirements
- iOS 15.0+, Xcode 15.0+
- iPhone 12 Pro or later for ProRAW

## Setup
```bash
cd ColorCorrection
pod install
```

See full documentation in project files.
