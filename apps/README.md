# Applications Directory

This directory contains all client applications for the Color Correction System.

## Structure

- **/mobile/android** - Android mobile app (Kotlin, Camera2/CameraX, OpenCV)
- **/mobile/ios** - iOS mobile app (Swift, AVFoundation, ProRAW/DNG)
- **/desktop** - Desktop professional app (Python, PyQt, OpenCV, RawPy)
- **/web** - Web portal (Next.js, React, TypeScript, Tailwind CSS)

## Features

### Mobile Apps
- RAW/DNG capture with manual controls
- Real-time color card detection overlay
- Image upload to cloud processing
- Low-resolution preview correction
- Notifications on job completion

### Desktop App
- Full professional workflow
- Batch processing up to 10k images
- Hot-folder and drag-drop import
- Advanced color correction controls
- Export with ICC profiles
- Lightroom plugin integration
